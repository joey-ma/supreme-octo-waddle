# supreme-octo-waddle
Exploring the C programming language.

## C
- general-purpose
- can be used to develop software like 
  - operating systems, 
  - databases, 
  - compilers, and so on.
- created in the 1970s by Dennis Ritchie
- by design, C's features cleanly reflect the capabilities of the targeted CPUs
